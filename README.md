# Station-Manager

RFS (Rural Fire Service) Station Management Application

## Overview

This application provides a digital sign-in system for RFS station members with real-time synchronization across multiple devices. Members can check in via kiosk iPads, mobile phones, or QR codes, with activity tracking for training, maintenance, meetings, and custom activities.
